faceRecognition module¶

class faceRecognition.FaceRecognition[source]¶

Bases: object

This class comprises of functionalities to recognise a persons face and helps registering and log into the system using facial recognition techniques.

capture()

Captures image using mounted camera and saves it to directory to later on recognize based on that.

encode()

Loads the known faces and embeddings. add each encoding and name to our set of known names and encodings.

recognise()

Recognises the known faces.
