class UserDB.UserDB[source]¶

Bases: UserDB.DataBase

UserDB class is a child class of abstract class DataBase class to implement the behaviors of the abstract methods in the abstract class also with some additiontial behaviors thrugh different added methods as checking if user exists in the database, if the password is correct, getting all user information from the database. …

It does not have an attribute it only contains methods as it is ensuring the implementation of the abstract methods.
