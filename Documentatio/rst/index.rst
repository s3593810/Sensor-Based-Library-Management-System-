Welcome to LMS's documentation!
===============================
This the documentation for the Final Project IoT, Semester 2, 2019.
This projecrt is an smart library system to allow users interact digitally 
to borrow, return or search for books.
Special Funtionality Includes: 

User can use QR scanner Mechanism to Borrow or Return a Book.
User can use voice Recognition to Search for Book.
User can use Facial Recognition Technologies to Login to the System.
